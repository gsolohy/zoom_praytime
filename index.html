Before You Download

  • This is for Windows OS - Not Mac OS
  • Zoom users must not change name
  • Zoom users must type in chat only when he/she 'log on' and 'log out' but nothing more than that

Download

  • Click: pray_timer_1.0.0.zip

Installation

  • Extract pray_timer_1.0.0.zip
  • "pray_timer.exe" and "pray_timer.py" must be in the same folder

How to Use "pray_timer.exe"

  • Save chat-log from zoom meeting
  • File name should be "meeting_saved_chat.txt" already, if not please rename it so

  • Place "meeting_saved_chat.txt" file in the same folder as "pray_timer.exe"

  • Run "pray_timer.exe" and wait a little...

  • "prayer_time.csv" will be created in the same folder
  • Open in excel and use it to your need
